# PromptCraft Edu

Esta é uma aplicação interativa para formação de professores na escrita de prompts eficazes para sistemas de IA.

## Descrição

O PromptCraft Edu foi desenvolvido para ajudar professores a aprender, praticar e aperfeiçoar habilidades de prompt engineering voltadas para o contexto educativo. A aplicação permite que os usuários:

- Aprendam princípios fundamentais para criação de prompts eficazes
- Pratiquem com cenários educativos realistas
- Recebam feedback imediato sobre seus prompts
- Progridam através de diferentes níveis de complexidade

## Funcionalidades

- **Introdução aos conceitos**: Visão geral da importância de prompts bem elaborados no contexto educativo
- **Guia de princípios**: Explicações e exemplos de cada princípio fundamental
- **Cenários práticos**: Desafios baseados em situações reais de ensino em três níveis de dificuldade
- **Sistema de avaliação**: Feedback detalhado sobre a qualidade dos prompts criados
- **Progressão adaptativa**: Desbloqueia níveis mais avançados conforme o usuário demonstra proficiência

## Como usar

1. Faça o download ou clone este repositório
2. Abra o arquivo `index.html` em um navegador web moderno
3. Navegue através das diferentes seções da aplicação:
   - Introdução: Visão geral do propósito da aplicação
   - Princípios: Aprenda os fundamentos da escrita de prompts eficazes
   - Prática: Aplique os princípios em cenários educativos reais

## Implementação técnica

A aplicação é desenvolvida com:
- HTML5
- CSS (com Tailwind via CDN)
- JavaScript (React via CDN)
- Lucide Icons para elementos visuais

## Hospedagem

Para hospedar esta aplicação no GitHub Pages:

1. Faça um fork deste repositório ou crie um novo
2. Faça upload dos arquivos para seu repositório
3. Ative o GitHub Pages nas configurações do repositório, selecionando a branch principal como source
